# Queue de Castor - Montreal Beaver Tails Web App

A simple web application showcasing "Queue de Castor" (Beaver Tails) - Montreal's beloved Canadian pastry.

## About

This web app provides information about Beaver Tails, a famous Canadian fried dough pastry shaped like a beaver's tail. The site includes:
- Information about what Beaver Tails are
- Menu of delicious flavors
- Location and contact information
- Fun facts about this iconic Canadian treat

## Installation

1. Clone the repository:
```bash
git clone https://github.com/nathanzbl/skill3.git
cd skill3
```

2. Install dependencies:
```bash
npm install
```

## Usage

Start the server:
```bash
npm start
```

The application will be available at `http://localhost:3000`

## Technologies Used

- Node.js
- Express.js
- HTML5
- CSS3

## Features

- Responsive design that works on mobile and desktop
- Beautiful gradient styling
- Interactive hover effects on menu items
- Information about the history and varieties of Beaver Tails
